Gravity

A framework for beautiful websites

"Gravity is a framework for building rock solid HTML5 websites. It uses SASS to control styles, so make sure you download it before trying this framework. All the typography in this example is set to a 20px baseline grid. Be sure to checkout the various sass mixins, found in lib/helpers/_mixins.scss"
